# Odin Recipes

Odin Recipes is a simple static website that showcases several cooking recipes.  
This project is part of **The Odin Project – HTML Foundations** curriculum and is intended to practice basic HTML structure.

## Project Purpose

The main goal of this project is to strengthen my understanding of:
- HTML boilerplate
- Headings and paragraphs
- Links and images
- Lists (ordered and unordered)
- Conventional Commit Message

## Built With

- HTML5

## What I Learned

Through this project, I learned how to:
- Structure a webpage
- Link multiple pages together
- Organize assets such as images properly
- Write conventional git commit message
